Graduates Hub has evolved from a South African-focused learning platform to a globally-oriented educational resource. ✅ WebSite + SearchAction

Signals Google to show a Sitelinks Search Box in the SERP.

"potentialAction": {
  "@type": "SearchAction",
  "target": "https://www.graduateshub.org/search?q={search_term_string}",
  "query-input": "required name=search_term_string"
}

✅ FAQPage Schema

Interactive FAQ components embed JSON-LD inline, making each FAQ eligible for Google's FAQ rich results. Used on: homepage, every guide page, category pages, every interview prep page.

✅ HowTo Schema on Career Roadmaps

Each roadmap stage maps to a HowToStep, including totalTime extracted from the "time to job-ready" field. Eligible for rich results in how-to queries.

✅ Person Schema with sameAs

Both founders (Jason Sadiki, Ndulamiso Mamburu) are described with full Person schema including LinkedIn and personal website sameAs links, contributing to Google's Knowledge Graph entity recognition.

4Issues & Optimization Opportunities

4.1 CRITICAL Issues

1. Course Schema Currency Hardcoded as ZAR

Location: /app/course/[courseId]/page.tsx line 46

Current: "priceCurrency": "ZAR"

Impact: Confuses international users in search results; suggests courses are only for South Africa

Recommendation:

  • Option A: Change to "USD" for global consistency
  • Option B: Remove currency entirely and use "Free"
  • Option C: Add currency based on user geo-location (requires server logic)

Priority: HIGH | Effort: LOW

4.2 MEDIUM Priority Issues

1. Missing Keywords Optimization

Issue: Pages lack explicit keywords meta tag (deprecated but still helpful as guidance)

Pages Affected: Most pages

Recommendation: Define target keywords for each page type:

  • Career roadmaps: "[Role] career path", "[Role] salary [Region]", "how to become [role]"
  • Interview prep: "[Role] interview questions", "[Role] interview tips", "[Role] salary"
  • Courses: "[Subject] free course", "learn [subject]", "[Subject] certification"

Priority: MEDIUM | Effort: MEDIUM

2. No hreflang Tags

Issue: Missing hreflang for language/region variants

Current State: Single English version only

Recommendation: If planning multi-language expansion, implement hreflang tags

Priority: MEDIUM | Effort: MEDIUM (only if expanding to multiple languages)

4.3 LOW Priority Issues / Opportunities

1. FAQPage Schema Not on All FAQ Pages

Pages Affected: Career roadmaps, Interview prep (have FAQs in HTML but schema could be added)

Benefit: Enables FAQ snippets in Google Search

Priority: LOW | Effort: MEDIUM

2. BlogPosting Schema Enhancement

Current: BlogPosting schema exists but could include more rich fields

Potential Additions: mainEntity (Topic), keywords, inLanguage

Priority: LOW | Effort: MEDIUM

3. Video Schema Not Implemented

Benefit: Could add for tutorial videos if future content includes video

Priority: LOW | Effort: MEDIUM
